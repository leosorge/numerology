# 🔮 Numerologia Antica & AI

Una web application premium costruita con **Python (Flask)** che esplora le vibrazioni numeriche del tuo nome e della tua data di nascita, potenziata dall'intelligenza artificiale di **Google Gemini**.

![Screenshot dell'App](https://via.placeholder.com/800x450?text=Numerologia+Antica+UI)

## ✨ Funzionalità

- **Calcolo Consonanti (Ego Esterno)**: Somma ASCII-64 ridotta alla radice digitale.
- **Calcolo Vocali (Anima Interna)**: Analisi delle vibrazioni emotive.
- **Calcolo Totale (Destino)**: La sintesi del tuo percorso di vita.
- **Calcolo Data (Percorso)**: Interpretazione del tuo giorno di nascita.
- **AI Text Architect**: Integrazione opzionale con Gemini AI per riscrivere e modernizzare i responsi in tempo reale.
- **Design Premium**: Interfaccia in Dark Mode con effetti Glassmorphism e animazioni fluide.

## 🚀 Installazione Locale

1. **Clona il repository**:
   ```bash
   git clone https://github.com/TUO-UTENTE/numerology.git
   cd numerology
   ```

2. **Installa le dipendenze**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Avvia l'applicazione**:
   ```bash
   python app.py
   ```

4. **Accedi**: Apri il browser su `http://127.0.0.1:5000`.

## 🤖 Integrazione AI

Per attivare la riscrittura dinamica dei testi:
1. Ottieni una API Key gratuita su [Google AI Studio](https://aistudio.google.com/).
2. Inserisci la chiave nell'apposito campo "Gemini API Key" nell'interfaccia dell'app.

## 🛠️ Tecnologie Utilizzate

- **Backend**: Python, Flask
- **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (Async/Fetch)
- **AI**: Google Generative AI (Gemini 1.5 Flash)
